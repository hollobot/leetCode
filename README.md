# leetCode

算法学习与刷题记录，使用 Java 实现 LeetCode 题目、周赛题解以及 ACM 练习。

## 环境

- JDK 17
- Maven
- JUnit 4.13.2

## 构建

```bash
mvn compile        # 编译
mvn test           # 运行测试
```

## 目录结构

源码位于 `src/main/java`，按算法主题分类组织：

| 目录 | 说明 |
| --- | --- |
| `com/example/backtracking` | 回溯 |
| `com/example/binarySearch` | 二分查找 |
| `com/example/bit` | 位运算 |
| `com/example/dp` | 动态规划（入门 DP、打家劫舍、最大子数组和、经典线性 DP 等） |
| `com/example/heap` / `queue` | 堆与优先队列 |
| `com/example/linkedList` / `listNode` | 链表 |
| `com/example/math` | 数学 |
| `com/example/sort` | 排序（归并排序等） |
| `com/example/stack` | 栈 |
| `com/example/string` / `str` / `字符串` | 字符串 |
| `com/example/tree` | 树（Trie、二叉树遍历与构造等） |
| `com/example/windowSlide` / `滑动窗口` | 滑动窗口（定长 / 不定长） |
| `com/example/区间` | 区间问题 |
| `com/example/图的广度优先搜索` | 图的广度优先搜索 |
| `com/example/数组` | 数组 |
| `com/example/数据结构` | 数据结构设计 |
| `com/example/简单` | 入门题 |
| `com/example/thread` | 多线程练习 |
| `com/example/io` | IO 与序列化 |
| `contest` | LeetCode 周赛题解（按场次划分） |
| `acm` | ACM 风格练习 |

## 说明

题目命名多采用中文题名，便于检索对应 LeetCode 题目。
